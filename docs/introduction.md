# Introduction
Lux is a component library for Laravel inspired by <a href="https://ui.shadcn.com/" target="_blank" rel="nofollow">Shadcn</a>. It integrates with Tailwind CSS v4, Alpine.js, and optionally Livewire to help you build dynamic, responsive applications.

## Features
- **Blade Components** — A growing collection of ready-to-use Laravel Blade components.
- **TALL Stack** — Built for Tailwind CSS, Alpine.js, Livewire, and Laravel. Livewire is optional.
- **Wireable** — All components support `wire:model` for seamless Livewire binding.
- **Customizable** — Easily adjust appearance and behavior to match your project.
